FLEET CAPACITY RADAR · STARLINK

Every satellite holds altitude only while its orbit-maintenance capacity exceeds atmospheric drag injection. When the ratio Θ crosses its threshold, the satellite leaves the envelope and begins the fall to the 120 km burn line. This board reads that state for the whole fleet from public orbital elements.

Validated early warning

The same latched detector this portfolio runs on Li-ion cells, bearings and gear run-to-fail data runs in two tiers, red and amber, both calibrated on 10 healthy Starlink satellites only and then applied unchanged to 21 real deorbits (December 2025 to June 2026, three per month), across 28,825 archived element sets.

Alarm rules, both latched and both set from the healthy cohort alone: red fires on loss above 2.9 km/day for 5 consecutive days; amber fires on loss above 0.05 km/day for 24 consecutive days, a persistence window set to twice the longest gentle-descent episode ever observed in a healthy satellite. The single-threshold baseline scored a 49-day median with the same protocol; the amber tier lifted the six weakest cases from 6 to 10 days of warning up to 149 to 247 days. Per-satellite table: starlink_earlywarning_results.csv in this repository.
Method. Semi-major axis and its decay rate are derived from each object's public two-line elements; the capacity-state classification applies the same Θ (injection over capacity) framework used by this portfolio's other instruments on hydraulic rigs, Li-ion cells, bearings and gear run-to-fail data. Honest scope: the board above reads the fleet's current state from a public snapshot; the early-warning layer is validated on archived orbital histories with a controls-only calibration protocol (threshold set on healthy satellites, never on the deorbits it was scored against). Most Starlink reentries are controlled descents, so the validated claim is robust, false-alarm-free detection of sustained loss of altitude-hold across the whole fleet with a measured runway to burn-up, not prediction of surprise failures. Raw element sets are not redistributed, per the Space-Track user agreement. Detector internals are not exposed. Built by Oren A. L. Speiser · public data only · no affiliation with SpaceX.
